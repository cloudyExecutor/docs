---
title: "Auto Inject"
description: "Sets Auto Attach"
---

<Note>
  This feature isn't really needed anymore, because we added a TP-Handler, which doesn't get dejected when leaving. However, it's still available with:
</Note>

```csharp
Api.SetAutoAttach(true);
```

you can disable it with:

```csharp
Api.SetAutoAttach(false);
```
