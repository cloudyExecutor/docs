## Attach the Api

You can use the `Attach` method like this:

```csharp
private void YOURBUTTON_CLICK(object sender, RoutedEventArgs e)
{
    Api.Attach();
}
```

here is a sample usage:

```csharp
private void inject_Click(object sender, RoutedEventArgs e)
{
    Api.Attach();
}
```
